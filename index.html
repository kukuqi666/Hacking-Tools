## Subdomain Enumeration Tools
  • **OneForAll**

    A powerful subdomain enumeration tool supporting multiple data sources.

  • **knock**

    A Python-based tool designed to enumerate subdomains on a target domain through a wordlist.

  • **subbrute**

    A DNS meta-query spider that enumerates DNS records and subdomains using multiple techniques.

  • **subdomain3**

    An efficient tool for finding subdomains, combining multiple techniques.

  • **Sublist3r**

    A Python-based tool designed to enumerate subdomains of websites using OSINT.

  • **dnsrecon**

    DNS reconnaissance tool with the ability to enumerate records and check various DNS-related activities.

  • **dnsenum**

    A multi-threaded Perl script to enumerate information on DNS domains.

  • **subDomainsBrute**

    A subdomain enumeration tool that uses a dictionary-based brute force approach.

  • **JSFinderPlus**

    A tool to find URLs and subdomains from JavaScript files.

  • **URLFinder**

    A simple and efficient tool to extract URLs from web pages.

  • **JSFinder**

    A tool for discovering URLs from JavaScript files for use in security assessments.
